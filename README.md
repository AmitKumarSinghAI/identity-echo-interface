# Identity Echo Interface

## MirAI School of Technology - Virtual Summer Internship 2026  
### AI Builder Track - Assignment 1

This is my first assignment completed during the **Virtual Summer Internship 2026 at MirAI School of Technology** under the **"AI Builder" Track**.

The project is called **"The Identity Echo Interface"**. It is a simple interactive web application built using **Streamlit** that collects user information, validates inputs, and generates a personalized response.

---

## Project Overview

The Identity Echo Interface demonstrates the basics of building an interactive web application using Streamlit.

The application allows users to:

- Enter their name
- Enter a personal message
- Submit the information using a button
- Receive a personalized transmission response
- View an estimated token consumption count for their message

---

## Features

✅ Streamlit-based user interface  
✅ User name input field  
✅ User message input field  
✅ Button-based execution control  
✅ Input validation using conditional statements  
✅ Error handling for missing name  
✅ Warning handling for missing message  
✅ Personalized success message using Python f-string  
✅ Token usage estimation based on character count
